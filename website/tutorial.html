How to use NCBI-RABV-GLUE - an example-driven tutorial

Before attempting this tutorial, you may first want to investigate the GLUE example project which provides a broad introduction to the GLUE framework.

Genomic approaches have great potential to help control the spread of RABV. In this tutorial we will use the NCBI-RABV-GLUE project to...

This tutorial comprises the following steps:

  1. Downloading RABV sequences from GenBank.
  2. Genotyping RABV sequences.
  3. Extracting isolate data from GenBank files.
  4. Submitting RABV sequences to GenBank.
  5. Filtering data to obtain a valid analysis data set.
  6. Refining the RABV reference sequence.
  7. Creating an alignment of all RABV sequences.
  8. Mapping feature coverage in all RABV isolates.
  9. Creating an automated procedure for reconstructing midpoint-rooted phylogenetic trees.
A brief section at the end of this tutorial discusses how GLUE resources can be collaboratively developed via online repositories such as GitHub.
